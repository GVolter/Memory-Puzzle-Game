# Memory Puzzle Game
![logo](./pics/logo_game.png)

A simple memory puzzle created using Pygame library. 
* It has 3 difficulties: Easy (3x4) , Medium (4x4) and Hard(4x5)
* Options to Play Again and Return to Main Menu
